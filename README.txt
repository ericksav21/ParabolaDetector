Para compilar:
-Usar 'make' sin comillas.

Para ejecutar:
-Usar './main' sin comillas.

Notas:
-Para más información sobre el makefile usar el comando 'make help' sin comillas.
-Ya existen configuraciones para corridas por defecto. Para ejecutarlas use los comandos 'make run1' y 'make run2' sin comillas, las cuales trabajan con el UMDA y el algoritmo genético.
-Para información sobre los parámetros que recibe el programa usar './main --help'.
